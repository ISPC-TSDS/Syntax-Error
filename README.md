# Syntax Hub

## Introducción

Syntax Hub es una plataforma web diseñada para la gestión de reservas de espacios de coworking. Su propósito es permitir que los usuarios puedan consultar las salas disponibles, seleccionar una fecha y horario, y realizar reservas de forma simple y organizada.

El sistema busca resolver la dificultad de administrar manualmente las reservas de salas, evitando superposiciones de horarios y mejorando la experiencia tanto para los usuarios como para los administradores.

---

## Funcionalidades del Proyecto

### Usuario
- Registrarse e iniciar sesión.
- Visualizar las salas disponibles.
- Seleccionar una sala.
- Seleccionar fecha y hora.
- Confirmar una reserva.
- Ver sus reservas.
- Cancelar reservas.

### Administrador
- Crear, editar y eliminar salas.
- Gestionar reservas.
- Gestionar usuarios.
- Visualizar reportes del sistema.

---

## Instalación y Ejecución

1. Clonar el repositorio:

git clone https://github.com/ISPC-TSDS/Syntax-Error.git

2. Ingresar al directorio del proyecto:
cd Syntax-Error/maqueta
3. Abrir el archivo home.html en cualquier navegador web.
